# Project Title: Insurance Data Analysis Dashboard

# Link : https://app.powerbi.com/groups/20236ac0-74a1-45d1-8ae7-363bc0b43c93/reports/4b07e279-7a5c-43ba-a9b3-5653f6bfc266/3c68bd31730ccc303713?experience=power-bi

# Description : 
This project analyzes insurance data to provide insights into bookings, customer ratings, and overall performance metrics using Power BI. The dashboard aims to enhance decision-making for insurance companies.

# Tech Stack Used :
1. Power BI: For data visualization and dashboard creation.
2. SQL: For data extraction and transformation.
3. Excel: For initial data manipulation and preparation.
4. DAX: For creating calculated columns and measures to enable complex analytics.

# Key Responsibilities :
1. Data Sourcing: Extract data from SQL databases and Excel files.
2. Data Transformation: Utilize Power Query Editor for data cleaning and transformation.
3. Dashboard Development: Build interactive visualizations showing key metrics related to insurance bookings and customer feedback.
4. Collaboration: Share dashboards with stakeholders and implement row-level security for data privacy.
5. Data Analysis: Analyze trends and patterns in customer ratings and insurance claims.

# Key Highlights :
1. Performance Metrics: Visuals depicting key performance indicators (KPIs) like claim approval rates, customer satisfaction scores, and booking trends.
2. Customer Feedback Analysis: In-depth analysis of customer feedback to identify areas for service improvement.
3. Comparative Analysis: Comparison of different vehicle types in terms of insurance bookings and ratings.

# Outcomes & Insights :
Identified significant trends in customer preferences regarding vehicle types and insurance packages.
Provided actionable insights to improve service offerings, resulting in higher customer satisfaction.
Developed a dashboard that allows for real-time data interaction, enabling stakeholders to make informed decisions quickly.
