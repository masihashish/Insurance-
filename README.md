# Project Title: Insurance Data Analysis Dashboard

# Description : 
This project analyzes insurance data to provide insights into bookings, customer ratings, and overall performance metrics using Power BI. The dashboard aims to enhance decision-making for insurance companies.

# Tech Stack Used :
Power BI: For data visualization and dashboard creation.
SQL: For data extraction and transformation.
Excel: For initial data manipulation and preparation.
DAX: For creating calculated columns and measures to enable complex analytics.

# Key Responsibilities :
Data Sourcing: Extract data from SQL databases and Excel files.
Data Transformation: Utilize Power Query Editor for data cleaning and transformation.
Dashboard Development: Build interactive visualizations showing key metrics related to insurance bookings and customer feedback.
Collaboration: Share dashboards with stakeholders and implement row-level security for data privacy.
Data Analysis: Analyze trends and patterns in customer ratings and insurance claims.

# Key Highlights :
Performance Metrics: Visuals depicting key performance indicators (KPIs) like claim approval rates, customer satisfaction scores, and booking trends.
Customer Feedback Analysis: In-depth analysis of customer feedback to identify areas for service improvement.
Comparative Analysis: Comparison of different vehicle types in terms of insurance bookings and ratings.

# Outcomes & Insights :
Identified significant trends in customer preferences regarding vehicle types and insurance packages.
Provided actionable insights to improve service offerings, resulting in higher customer satisfaction.
Developed a dashboard that allows for real-time data interaction, enabling stakeholders to make informed decisions quickly.
